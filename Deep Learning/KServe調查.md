# KServe
什麼是 KServe？

- Kubernetes 的 (ML, DNN)  Model Inference Platform 框架
- 可自訂資源、快速部屬、高吞吐能力
- 兼容多種ML, DNN框架 (Tensorflow, XGBoost, ScikitLearn, PyTorch, and ONNX)
- ML deployments 特色:
	- GPU Autoscaling
	- Scale to Zero
	- Canary Rollouts
- 支援 serverless inference workload
